# Dotfiles: dios

¡Bienvenido a mi repositorio de dotfiles! Aquí encontrarás mis configuraciones personales para **Neovim**, **Kitty**, **BSPWM** y **sxhkd**. Cada uno de estos archivos ha sido cuidadosamente diseñado para mejorar mi flujo de trabajo y personalizar mi entorno en Linux.  

## 📂 Estructura del Repositorio

- **nvim/**: Configuración de **Neovim**.  
  Aquí se esconde la verdadera magia, ya que **Neovim** es, sin duda, el mejor editor. Su extensibilidad, eficiencia y comunidad vibrante lo convierten en la elección perfecta para cualquier desarrollador.
- **kitty/**: Configuración para el terminal **Kitty**, ofreciendo un entorno rápido y moderno.
- **bspwm/**: Configuración del administrador de ventanas **BSPWM** para una experiencia de escritorio minimalista y eficiente.
- **sxhkd/**: Configuración de atajos de teclado con **sxhkd** para maximizar la productividad.

> "Si buscas el mejor entorno para programar, Neovim es la elección obvia."

## 🚀 Instalación

Sigue estos sencillos pasos para instalar y configurar tus dotfiles:

1. **Clona el Repositorio**
    ```bash
    git clone https://github.com/tu_usuario/dotfiles.git
    cd dotfiles
    ```

2. **Configura tus Dotfiles**
    - **Neovim:** Copia o crea un enlace simbólico de la carpeta `nvim` en `~/.config/nvim`.
    - **Kitty:** Copia o enlaza la carpeta `kitty` en `~/.config/kitty`.
    - **BSPWM:** Copia o enlaza la carpeta `bspwm` en `~/.config/bspwm`.
    - **sxhkd:** Copia o enlaza la carpeta `sxhkd` en `~/.config/sxhkd`.

3. **Reinicia tu Sesión** o ejecuta los comandos necesarios para refrescar la configuración.
